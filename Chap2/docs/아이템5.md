해결책

- 싱글턴이나 정적 유틸리티 클래스를 사용하는 대신에 클래스가 여러 인스턴스를 지원해야 하고 클라이언트가 원하는 자원을 사용할 수 있도록 설계해야 한다. 따라서 인스턴스 생성시 생성자에 필요한 객체를 넘겨주어 의존 객체를 주입한다.
